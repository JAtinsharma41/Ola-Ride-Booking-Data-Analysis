# Ola-Ride-Booking-Data-Analysis


## Project Overview

This project analyzes Ola ride booking data using SQL to identify ride demand patterns, cancellation trends, revenue insights, and customer behavior.

## Dataset

The dataset contains 100,000 ride booking records with details such as:

* Booking ID
* Customer ID
* Vehicle Type
* Ride Distance
* Booking Status
* Driver Rating
* Customer Rating
* Payment Method

## Tools Used

* SQL Server
* Excel
* GitHub

## Key Analysis

* Most used vehicle type
* Average ride distance
* Driver and customer ratings
* Revenue from successful rides
* Ride cancellation reasons

## Files in Repository

* `Bookings-100000-Rows.xlsx` → Dataset
* `ola_sql_analysis.sql` → SQL queries used for analysis
* `README.md` → Project documentation
